
Folder used in creating an installation release
	_Release			

Folders contain MS VS Fortran projects for executables run by SCILAB
	ArgonneBoxes		Integrates the spot intensities
	GenHKLs				Creates a list of HKLs
	ImageInfo			Reads TIF header to extract information about a data file
	IndexConic			Used to index images using conics
	Laue4				Merging and normalising spot intensities
	LauegCyclops		Converts old Cyclops files to use with LaueG
	MakeEllipses		Used to display integration ellipses for an image file
	OrientSpots			Refines the crystal orientation once indexed
	OrientTwins			Many versions trying to orient images with twins (unfinished)
	RejectsData			Searches *.ell files for the elllipse of the "reject spots"
	RejectsImage		Create images used for rejecting outlier spots
	StripBack			Extract the background from an image (doesn't work well with temporary shielding)

Folder contains a MS VS fortran project, plus registry file
	LaueTiff			A file explorer utility to display info from the TIF data files using a right-click
